Date Calc

Enter a date using any format below:

  • yyyy-mm-dd: 2017-01-28
  • mm/dd/yyyy: 01/28/2017
  • mmm dd yyyy: Jan 28 2017
  • mmmm dd yyyy: January 28 2017
  • today

Or specify a date using the URL query string p1:

https://zzpxyx.github.io/date-calc/index.html?p1=today
